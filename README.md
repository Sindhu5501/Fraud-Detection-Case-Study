# Financial Fraud Detection Case Study
This project analyzes a 6.3M+ transaction dataset to identify fraudulent activity using Machine Learning.

## Key Features
- **Data Scaling:** Optimized memory usage for 6.3M+ rows.
- **Feature Engineering:** Created `errorBalance` features to catch account discrepancies.
- **Model:** Random Forest Classifier.
- **Evaluation:** Focused on **AUPRC** and **Recall** due to class imbalance.
